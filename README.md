# DropOut
本项目是 [yui540/DropOut](https://github.com/yui540/DropOut) 的 Fork 版本。

## 项目改进
- 迁移到现代构建工具 Vite
- 将网页内容翻译成中文
- 优化项目结构和配置

## 超过50秒的震撼CSS动画
本网站的动画部分全部使用CSS Animation实现。

支持PC、平板、智能手机无缝运行（如果设备能承受渲染处理的话）
